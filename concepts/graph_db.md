## What Are Text Notes?
Text notes are digital documents that contain unformatted, plain text. They lack rich media, complex layouts, or custom fonts, prioritizing speed, simplicity, and cross-platform compatibility. [1, 2, 3, 4, 5]
## Core Formats

* Plain Text (.txt): The universal standard. It contains zero formatting and opens on virtually any device or operating system.
* Markdown (.md): A lightweight syntax that adds formatting (like bolding, headers, and bullet points) to plain text using keyboard symbols. [6, 7, 8, 9, 10]

## Key Advantages

* Universal Compatibility: Open and edit files on Windows, macOS, Linux, iOS, or Android without losing formatting.
* Future-Proof: Plain text files do not rely on proprietary software, ensuring data is readable decades from now.
* High Performance: Files require minimal storage space and load instantly, even on older hardware.
* Searchability: Modern operating systems can instantly index and search thousands of text notes. [11, 12, 13, 14, 15]

## Popular Text Note Apps

* Obsidian: Uses Markdown files stored locally on your device. It links notes together like a personal web, mimicking a graph database structure.
* Logseq: A privacy-first, open-source outliner that utilizes local Markdown or Org-mode text files.
* Notepad / Apple Notes: Built-in operating system defaults tailored for quick, distraction-free capture.
* VS Code / Vim: Power-user text editors equipped with extensions for advanced note organization. [16, 17, 18, 19, 20]

## Common Use Cases

* Quick Capture: Jotting down temporary thoughts, phone numbers, or checklists.
* Journaling: Writing daily logs in a structured, long-term format like Markdown.
* Knowledge Management: Building a personal wiki (Zettelkasten) by linking plain text files together.
* Coding/Scripting: Drafting snippets of code, configurations, or system documentation. [21, 22, 23, 24, 25]
